RouterRoute
=======

Маршрут на роутере. Маршрутизация процесс сетевого уровня, определяющий лучший путь доставки пакета информации
получателю. Пути передачи пакетов информации называют маршрутами.
Маршруты обычно используются для передачи пакетов из одной сети в другую.
Для создания маршрута необходимо выбрать CIDR сети до которой будет осуществляться
маршрутизация и шлюз с помощью которого трафик будет в эту сеть поставляться, как
правило шлюз должен иметь доступ к обоим сетям


Объект "Маршрут на роутере"
----------------------

.. autoclass:: esu.RouterRoute


Примеры использования
---------------------

Создать новый маршрут на роутере:

.. code-block:: python

  from esu import Router, RouterRoute

  router_id = '58385696-32c6-4a5c-bafe-895815eedf04'
  router = Router.get_object(router_id)
  router_route = RouterRoute(router=router, destination="10.0.2.0/24",
                             nexthop="10.0.1.1")
  router_route.create()
